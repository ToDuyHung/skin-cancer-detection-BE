## Installation
### Install FastAPI
`pip install fastapi`
### Install uvicorn
`pip install uvicorn`
### Install watchgod for advanced auto-reloading
`pip install watchgod`
### Install gdown
`pip install gdown`
### Install cv2 and imutils
`pip install imutils opencv-python`
### Install sklearn
`pip install sklearn`

## Run the server
`python main.py`
